Using the Licensing Web Site

The ITT Visual Information Solutions licensing web site allows you to retrieve your license quickly and easily. All of the information you will need to use the licensing web site will appear in your browser; when you have finished the license activation process the license will (in most cases) appear in the license wizard automatically. This help topic discusses some situations that may disrupt the normal process of using the licensing web site.

My browser didn't launch

The license wizard attempts to launch your web browser automatically, forwarding appropriate information about your computer. If your browser does not launch automatically, and display the licensing login screen, do the following:

  1. Launch your web browser manually.
  2. On the license wizard, click the Back button.
  3. Select the Request a license if you don't have internet access radio button and click the Next button.
  4. Copy the full URL shown under "Option 1" in the license request text file that appears, and paste it into your browser.

If you are on a UNIX system, you can also follow the steps described in UNIX: Web Browser Does Not Launch.

I don't have a login

If you do not have an ITT Visual Information Solutions login, click the Create new user account radio button on the licensing web site login screen.

I've forgotten my password

If you have an ITT Visual Information Solutions login but have forgotten the password, click the Forgot Password link on the licensing web site login screen.

My license was not automatically retrieved

If you have worked through the license activation process on the licensing web site but your license is not automatically inserted into the license wizard's license entry screen, do the following:

  1. On the license wizard, click the Back button.
  2. Select the Install a license you have received radio button and click the Next button.
  3. Select the Activated license radio button or the Evaluation license radio button, depending on the type of license you have generated on the licensing web site.
  4. Copy the license information from your browser window and manually paste it into the license wizard.
