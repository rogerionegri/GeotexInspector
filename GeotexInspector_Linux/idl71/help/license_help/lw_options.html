License Wizard Options

This screen allows you to choose what you would like the license wizard to do. Choose one of the following by selecting the appropriate radio button and clicking Next.

Activate a license using the internet

Select this option to request a license from the ITT Visual Information Solutions licensing web site. If you are a current customer, or have received a license number for your product, this web site will allow you to retrieve license information immediately.

After you click Next, the license wizard will attempt to launch a web browser and connect to the ITT Visual Information Solutions licensing web site. If you generate or retrieve a license from the licensing web site, the license wizard will automatically display the license entry screen. If you are unable to generate or retrieve a license from the licensing web site, or if you are unable to connect to the licensing web site for any reason, click Exit to close the license wizard.


Note
The license wizard attempts to launch your web browser automatically. If your browser does not launch, see Troubleshooting.

If are unable to generate or retrieve a license from the licensing web site from the machine on which you are installing the ITT Visual Information Solutions product, choose Request a license if you don't have internet access.

Request a license if you don't have internet access

If the machine on which you are installing your ITT Visual Information Solutions product does not have internet connectivity, you can create a text file containing your license request. After you click Next, the license wizard will display a text widget containing a partly completed license request. Follow the instructions in the text widget and select File → Save As to save the license request file on your computer.

After saving your license request file, click Exit to close the license wizard.

Install a license you have received

If you have received a license as a text file, you can enter the license using the license wizard. After you click Next, the license wizard will display the license entry screen. See Entering a License for additional details.

Learn how to connect to an existing license server

If a license server for your product is already running on your organization's local network, you can connect to that license server without generating or retrieving a new license file for your machine. After you click Next, the license wizard will display the Connecting to an Existing License Server help topic.

After reading the instructions, click Exit to close the license wizard.
