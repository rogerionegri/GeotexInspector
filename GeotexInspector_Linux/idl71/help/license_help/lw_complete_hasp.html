Installing the HASP Drivers

This screen displays the location on your computer where your license file was saved.

Your license requires that a HASP driver be installed on your system. In addition, your license may require that a license server be installed and running before your product will be properly licensed. Options on this screen allow you to install the HASP driver and optionally to install and start a license server on your local computer.

  • If you are installing your ITT Visual Information Solutions product for the first time, see To Install and Start a Local License Server.
  • If you are updating an existing license on an existing local license server, see To Restart an Existing Local License Server.

    • Note
    If, after completing the licensing process, your product does not run in licensed mode, consult the Installation and Licensing Guide in your product's online help system for additional licensing information.

To Install the HASP Driver

The Install HASP device driver check box is checked by default. Unless you know that you already have a HASP driver installed and do not want to install the version included with your ITT Visual Information Solutions product, leave this box checked.

To Install and Start a Local License Server

If your license requires a license server, check boxes that allow you to install and start the server will be visible. If you are installing your ITT Visual Information Solutions product for the first time, do the following:

  1. Check the Install license manager service check box.
  2. Check the Launch lmtools utility check box.
  3. Click Next.
  4. In the lmtools utility, select the Stop/Read/Reread tab.
  5. Click Start Server.
  6. Select File → Exit to close the LM Tools utility.

To Restart an Existing Local License Server

If your license requires a license server, and you are updating an existing license on an existing local license server, you must shut down and restart the license server before your updated license will be recognized. To restart the license server, do the following:

  1. Check the Launch lmtools utility check box.
  2. Click Next and then Finish..
  3. In the lmtools utility, select the Stop/Read/Reread tab.
  4. Click Stop Server.
  5. Click Start Server.
  6. Select File → Exit to close the LM Tools utility.
