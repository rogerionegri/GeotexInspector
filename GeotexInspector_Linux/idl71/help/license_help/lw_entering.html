Entering a License

This screen allows you to enter your license information. When you click Next, the license information will be saved in the appropriate location on your system.


Note
If you request a license via the web, this screen may be automatically populated with the retrieved license information. All you need to do is click Next.

License Types

Two types of licenses are available for ITT Visual Information Solutions products:

  • There are several types of activated license; the most common are node-locked, floating, and flexible single user licenses. Activated licenses consist of several lines of text in a single block.
  • An evaluation license uses a simplified format that includes a product name, an expiration date, and a license key.

Entering an Activated License

If you have an activated license, do the following:

  1. Copy the entire text of the license file and paste it into the text field. Do not edit the text.

    2. Alternately, click Import license text and select the license file from a location on your local network.

  2. Click Next.

Entering an Evaluation License

If you have an evaluation license, do the following:

  1. Select the product you are licensing from the Product pulldown menu.
  2. Enter the expiration date of the license in the Expiration text field.
  3. Enter the license key in the Key text field.
  4. Click Next.
