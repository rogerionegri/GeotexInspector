Installing and Starting the License Server

Your license requires that a license server be installed and running before your product will be properly licensed. Options on this screen allow you to install and start a license server on your local computer.

  • If you are installing your ITT Visual Information Solutions product for the first time, see To Install and Start a Local License Server.
  • If you are updating an existing license on an existing local license server, see To Restart an Existing Local License Server.

    • Note
    If, after completing the licensing process, your product does not run in licensed mode, consult the Installation and Licensing Guide in your product's online help system for additional licensing information.

To Install and Start a Local License Server

If you are installing your ITT Visual Information Solutions product for the first time, you must install and start a license server. Do the following:

Windows

  1. Check the Install license manager service check box.
  2. Check the Launch lmtools utility check box.
  3. Click Finish.
  4. Wait for the license server installation to complete.
  5. Select Start → Program Files → ITT Product → LM Tools to launch the LM Tools utility.
  6. Select the Stop/Read/Reread tab.
  7. Click Start Server.
  8. Select File → Exit to close the LM Tools utility.

UNIX/Macintosh

  1. Check the Install license manager check box.
  2. Check the Start license manager check box.
  3. Wait for the license server installation to complete.
  4. Click Finish.

On UNIX and Macintosh systems, output from the license manager is written to a file named lmgrd_log.txt in the license directory of the product installation directory.

To Restart an Existing Local License Server

If you are updating an existing license on an existing local license server, you must shut down and restart the license server before your updated license will be recognized. To restart the license server, do the following:

Windows

  1. Check the Launch lmtools utility check box.
  2. Click Finish.
  3. In the lmtools utility, select the Stop/Read/Reread tab.
  4. Click Stop Server.
  5. Click Start Server.
  6. Select File → Exit to close the LM Tools utility.

Unix/Macintosh

  1. Check the Start license manager check box.
  2. Open a shell window.
  3. Change directories to the directory in which you installed your ITT product. If you chose the default installation location, the directory is 
    4. /usr/local/itt

    on UNIX systems and 

    /Applications/ITT

    on Macintosh systems.

  4. Enter the following commands at the shell prompt: 
    5. ./idl70/bin/lmdown

    ./idl70/bin/lmgrd
  5. Click Finish.
